# Installation (установка)🔫
**termux**
```
curl https://raw.githubusercontent.com/lkqas/god-bomber/master/install.sh | sh
```
**Restart termux (перезапустите termux)**
```
godbomber
```
**Uninstall (удаление)**
```
rm -rf ~/.gb && sed -i '$ d' ~/../usr/etc/bash.bashrc
```
